Web GIS Final Portfolio

This final portfolio presents three revised Web GIS products created during the semester. The portfolio highlights interactive mapping, spatial data visualization, web-based cartographic design, and user-centered improvements based on course concepts and TA feedback.

Portfolio Overview

The three projects in this portfolio demonstrate different approaches to Web GIS: a JavaScript-based Mapbox map, an ArcGIS Online spatial analysis app, and an online spatial survey. Together, these products show how interactive mapping tools can support exploration, geographic analysis, and location-based data collection.

I selected these projects because they represent different technical skills from the course, including custom web map design, spatial analysis, user interaction, and public-facing geospatial communication.

Project 1: Mapbox Hotel Map Around UMass Amherst

Purpose

This Mapbox project presents selected hotel options around UMass Amherst. The map is designed to help users compare hotel locations in Amherst and Hadley.

Interactive Features

Users can click hotel points to view detailed pop-up information, use area-based filters, read hotel labels, and navigate the map with zoom and pan controls.

Revisions and Improvements

I revised this project by replacing the default basemap with a cleaner light basemap, adding category-based hotel colors, improving the legend, expanding pop-up information, adding hotel labels, and creating an area filter for Amherst and Hadley.

Course Concepts Applied

This project applies JavaScript web mapping, custom cartographic styling, interactive pop-ups, user filtering, responsive design, and clear visual hierarchy.

Project 2: La Fortuna Hotel Accessibility and Density Analysis

Purpose

This ArcGIS Online project analyzes hotel distribution and accessibility around La Fortuna, Costa Rica. The map includes hotel locations, hotel density, drive time areas, and Arenal Volcano as a geographic reference point.

Interactive Features

Users can explore the map through pop-ups, legend information, zooming, panning, and layer-based visual interpretation in an ArcGIS Online Instant App.

Revisions and Improvements

I revised this project by simplifying overlapping polygon layers, renaming layers with clearer titles, improving the legend, adjusting transparency, and revising the color scheme. I also revised the explanation so the analysis matches the visible map outputs and avoids unsupported claims about slope or urbanization.

Course Concepts Applied

This project applies web-based spatial analysis, density visualization, accessibility mapping, cartographic generalization, layer organization, and clearer map communication.

Open ArcGIS Online Project

Project 3: Online Spatial Survey

Purpose

This survey project collects location-based information through an online form. It demonstrates how Web GIS can be used for public participation and spatial data collection.

Interactive Features

Users can complete the survey online, answer structured questions, and submit spatial information using a map-based location question.

Revisions and Improvements

I revised this project by improving the survey title and description, clarifying the purpose of the survey, checking public sharing settings, and making the spatial data collection component easier to understand.

Course Concepts Applied

This project applies volunteered geographic information, web-based survey design, spatial data collection, usability, public participation GIS, and accessible public sharing.
